## Install

```sh
pnpm i @simplyhexagonal/package

# or
yarn add @simplyhexagonal/package

# or
npm install @simplyhexagonal/package
```

## Usage

```ts
import Package from '@simplyhexagonal/package';

const package = new Package();
```
